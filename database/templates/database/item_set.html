Congratulations!

You have succesfully created an item set!
To use the item set, you need to download it by clicking on the Download JSON button below and put it in your

League of Legends\Config\Global\Recommended\

folder to use it on any champion or

League of Legends\Config\Champions\{CHAMPION}\Recommended\

folder to use it on a specific champion.

You can view the JSON yourself by clicking the View JSON button. You can then copy paste that into a new file in the folders mentioned above to use it in-game.
